# 概要

Rust で AtCoder に挑む際の環境構築の手順と、私個人の設定を色々まとめました。

# 環境構築

## Rustup のインストール

:::message
Mac における手順です。
以下それぞれインストール済みの状態を前提としています。

- VSCode
- Homebrew

:::

`rustup` と併せて rust のパッケージ管理ツールである `cargo` および、
rust コンパイラの `rustc` もインストールされます。

```shell
brew install rustup-init # rustup-initをインストール
rustup-init # rustup-initを実行
```

## cargo-compete のインストール

```shell
cargo install cargo-compete
```

## ジャッジに合わせた Rust のインストール

ジャッジ更新によってバージョンが変動するため、

適宜 AtCoder 公式の言語一覧をご参照ください。

https://img.atcoder.jp/file/language-update/language-list.html

```shell
rustup install 1.70.0
```

# その他初期設定

## AtCoder のログイン情報の入力

```shell
cargo compete login atcoder
# IDとパスワードが聞かれるので、ご自身のものを入力します。
```

## AtCoder 用のファイル群の作成

```shell
cargo compete init atcoder
Do you use crates on AtCoder?
1 No
2 Yes
3 Yes, but I submit base64-encoded programs
1..3: # お好きなものを選択します。各クレート群を使用する場合、2を選択します。
```

`2`を選択した場合、以下のファイル群が作成されます。

```
./
│  compete.toml
│  template-cargo-lock.toml
│
└─.cargo
        config.toml
```

### `[test]`

`toolchain` のバージョンを現在のジャッジシステムに合わせます。

```toml
[test]
toolchain = "1.70.0"
```

### `[submit]`

language_id を `"5054"` に変更します。

```toml
[submit]
kind = "file"
path = "{{ src_path }}"
language_id = "5054"
```

### `[template]`

後述の `cargo compete new`時に生成される`.rs`ファイルの初期状態です。

お好きに編集してください。

例

```toml
[template]
src = '''
#[allow(unused_imports)]
use itertools::{iproduct, Itertools};
#[allow(unused_imports)]
use num_traits::pow;
#[allow(unused_imports)]
use proconio::{
    fastout, input,
    marker::{Chars, Usize1},
};
#[allow(unused_imports)]
use std::cmp::{max, min};
#[allow(unused_imports)]
use std::collections::{HashMap, HashSet, VecDeque};
#[allow(unused_imports)]
use std::iter::FromIterator;

#[fastout]
fn main() {
    input!{
        h: usize, w: usize,
        s: [Chars; h],
        mut plan: [(usize, usize, usize); h]
    }
}
'''
```

## cargo-compete の更新

cargo-compete を更新する場合は

上書きインストールすることで対応します。

```shell
cargo install cargo-compete --force
```

もしくは、cargo-update を使用します。

```shell
cargo install cargo-update
cargo install-update cargo-compete
```
